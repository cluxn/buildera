    Create AnimatedCounter.tsx as a "use client" component:
    - Import useEffect, useRef from react
    - Import useMotionValue, useSpring, useTransform, useInView, useReducedMotion, motion from motion/react
    - Props interface: target (number), suffix (string, default ""), delay (number, default 0)
    - Create ref for the span element (useRef HTMLSpanElement)
    - Get isInView using useInView(ref, { once: true })
    - Get prefersReducedMotion using useReducedMotion()
    - Create count motion value starting at 0 using useMotionValue(0)
    - Create spring from count with stiffness 60, damping 20 using useSpring
    - Create rounded display value using useTransform: format as Math.round(v) + suffix
    - In useEffect gated on isInView: if prefersReducedMotion, call count.set(target) immediately; otherwise use setTimeout with delay * 1000 then count.set(target)
    - Dependencies array: [isInView, prefersReducedMotion] — count, target, delay are intentionally excluded (motion values are not React state)
    - Render motion.span with ref, className text-[3.75rem] font-bold text-white, children = rounded
