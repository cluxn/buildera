Create each controller in app/Http/Controllers/Api/:

**SettingsController.php**
  Method: index(): JsonResponse
    $settings = \App\Models\Setting::all()->pluck('value', 'key');
    return response()->json($settings);
  (Returns all settings as flat key→value object)

**BlogPostController.php**
  Method: index(): JsonResponse
    $posts = BlogPost::with('author')->published()->latest('published_at')->get();
    return response()->json($posts);
  Method: show(string $slug): JsonResponse
    $post = BlogPost::with('author')->published()->where('slug', $slug)->firstOrFail();
    return response()->json($post);
  Return 404 if not found (firstOrFail throws ModelNotFoundException → Laravel returns 404)

**CaseStudyController.php**
  Method: index(): JsonResponse — same published pattern, optional ?industry query param filter
    If request()->has('industry'): ->where('industry', request('industry'))
  Method: show(string $slug): JsonResponse

**GuideController.php**
  Method: index(): JsonResponse — published, latest published_at, optional ?category filter
  Method: show(string $slug): JsonResponse

**TestimonialController.php**
  Method: index(): JsonResponse

    Optional filter: ?service= or ?industry= for tagged filtering
    Return response()->json($testimonials);
  (No show() needed — testimonials are always listed, no detail page)

**SeoMetaController.php**
  Method: show(string $type, string $slug): JsonResponse

    return response()->json($meta); // null is fine — frontend handles fallback

**RedirectController.php**
  Method: index(): JsonResponse

**NavItemController.php**

    return response()->json($items);

**FooterLinkController.php**
  Method: index(): JsonResponse

    return response()->json($links);
